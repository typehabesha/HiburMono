# Hibur Mono (ኅቡር ሞኖ)

Hibur Mono is a a monospaced styled font for the Ge'ez (Ethiopic) script. 
Latin letters of Hibur Mono have been borrowed from the Noto Sans Mono font.
Many of the Hibur Mono geometric properties are also conforming to the
Noto Sans Mono design.

The typeface is designed for use where columnar displays of letters is
advantageous.  For example, when developing application code and working
in operating system terminals.


## Cavaets

Many letters will appear slightly different from their regular shapes.
Letters and vowel marks will be fully recognizable but may appear more compressed than usual.
This is a necessity that comes from the fixed-width contraint of a monospaced typeface.
We have done our best to maximize the visual clarity of all letters, punctuation,
and numerals under these exteme conditions.  

Any suggestions for improvements are most welcome. Please use the project's [issue system](https://github.com/typehabesha/HiburMono/issues)
for submitting suggestions and any bugs or other defects encounters.
